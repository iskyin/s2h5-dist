# 文档运行环境

基于vuepress构建，相关API查看vuepress文档 (opens new window)

文档目录：docs/

文档运行命令：

# 安装依赖
yarn

# 运行
yarn dev

# 构建
yarn build

# 创建Markdown规范

  • 组件文档在guide/文件夹下添加Markdown文档
  • 在配置文件.vuepress中的config.js中配置添加相应的文件名
  • 文件名命名尽量简洁
