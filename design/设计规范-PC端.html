# ··· PC端规范

# 简介

  • 适用于财经系统PM、设计师、前端
  • 设计稿基于通用 Web 尺寸 1440*900px 像素， 分辨率 72pxt
  • 设计稿中文字体：微软雅黑 英文字体：Arial,开发时替换成相应系统默认字体

# 颜色

主色：#3a84fb — 主要用于头部、按钮、重要性文字和icon

辅色：#246bdd — 用于导航栏、个别重点性文字、icon

辅色：#3e96ff — 用于部分按钮、数据图表颜色、部分icon

辅色：#739bff — 用于部分按钮、数据图表颜色、部分icon

辅色：#e4eeff — 用于表单色块区分、部分icon

辅色：#f5f5f5 — 用于背景色、输入框、划分内容区域

辅色：#dddddd — 用于未输入提示性文字失效按钮、icon、分割线等

辅色：#7d7d7d — 用于文字信息名称、描述性文字等

辅色：#3c3c3c — 用于重要级文字信息、内页标题信息

辅色：#ff7474 — 用于移除、删除、错误提示、取消等内容

辅色：#ffb24f — 用于部分按钮、数据图表颜色、部分icon

辅色：#67d36f — 主要用于头部、按钮、重要性文字和icon

辅色：#36c2cf — 主要用于头部、按钮、重要性文字和icon

# 重要按钮

执行重要操作，一般位于屏幕下方或跟随主内容

# 功能按钮

带有功能操作的范围使用，一般位于表格右上角，表单填写右下角

# 特殊按钮

带有小部分功能操作的范围使用，一般位于导航栏

# 弹框

弹层梦半层填充颜色#000000，透明度为40%，宽度为700px，导读根据实际内容调整，圆角8px（弹框内需增加表单填写、表格等内容过多时，宽度为1000px）
