# 摄影-白平衡

# 白平衡

白平衡，字面上的理解是白色的平衡。其实准确来说，应该是色彩平衡。

相机的白平衡控制，是为了让实际环境中白色的物体在你拍摄的画面中也呈现出“真正”的白色。 不同性质的光源会在画面中产生不同的色彩倾向，比如说，蜡烛火焰的光线会使画面偏橘黄色，而黄昏过后的光线则会为景物披上一层蓝色的冷调。但是人类的视觉系统会自动对不同的光线作出补偿，所以无论在暖调还是冷调的光线环境下，我们看一张白纸永远还是白色的。

但相机则无法做到人类视觉系统那样的智能化，它无法自动做出补偿，它只会直接记录呈现在它面前的色彩，这就会导致画面色彩出现偏色，或者偏暖，或者偏冷。

# 色温

所谓色温，从字面解就是「颜色的温度」。 色温数值越低越偏向红色（愈暖），数值越高侧越偏向蓝色（愈冷）。

因为光线会造成颜色产生偏差。所以很多时候，我们就要来设置相机，让相机拍摄出真正的白色。

色温较暖的环境下，使用较冷的设定来呈现正确的白色色温

较冷的环境下，使用较暖的设定来呈现正确的白色

以下是一些色温的常见实例：

  • 16000-20000K：天空碧蓝的天气
  • 8000K：浓雾弥漫的天气
  • 6500K：浓云密布的天气
  • 6000K：略有阴云的天气
  • 5500K：一般的日光，电子闪光灯
  • 5200K：灿烂的正午阳光
  • 5000K：日光，这是用于摄影、美术和其他目的专业灯 箱的最常用标准
  • 3200K：日光灯
  • 2800K：钨丝灯／电灯泡（日常家用灯泡）
  • 1800K：烛光
  • 1600K：日出和日落

一般来说，数码相机有三种方法去获得正确的白平衡，分别为全自动／半自动及手动。随着技术的发展，大多数时候自动白平衡就可以满足需要了。

# 自动白平衡

让相机根据自身算法进行计算拍摄现场的色温，相机会自动作出最适当的调整。基本满足日常需要，光线过于复杂可能会出现错误。

# 日光模式

在正午阳光下的拍摄模式，这种模式下相机不需要任何调整，这种条件下的色温为非常理想的5000-5600K。在晚间拍摄夜景的人造光线时亦适用。

# 闪光灯模式

如果用闪光灯拍摄时，会造成画面看上去偏蓝，用这个模式可以解决这个问题。闪光灯色温偏「冷」一点，这种模式下相机会自动调为「暖」一些。

# 钨丝灯模式

钨丝灯胆会造成画面偏黄，室内拍摄常用模式。室内的钨丝光线色温偏「暖」一点，这种模式下相机会自动调为「冷」一些。

# 光管模式

不少光管的光线都会让相片色调在拍摄时变得蓝蓝绿绿, 这时可以用此模式。

# 阴天模式

阴天时拍照，光线会过度偏冷。这个模式可以增加暖调。

# 阴影模式

加强版的阴天模式。

# 自定模式

全手动模式，难度较大，由拍摄者自行设定相机的K值。

自定白平衡：

一、可以拿一张18% 灰卡（可以某宝购买），先拍摄一张有灰卡的照片，然后可以后期通过PS或者LR进行修正，只需要根据有灰卡这张照片取灰卡点进行自动白平衡修正即可。

二、现场找一张白纸，然后用相机拍摄白纸，然后将白纸调整为完全的白色，即可修正白平衡。

如果打算采用自定模式，建议采用方法一，方法二比较麻烦，且不一定准确。

比较暖的色温：例如想呈现出比较暖、比较温馨、怀旧、故乡的味道等，都可以使用较暖的色温（小Ｋ值）

比较冷的色温：例如想呈现出比较冷、比较高科技、近未来的感觉，都可以使用较冷的色温（高Ｋ值）
