# 快速上手

# S2H5-UI 简介

s2h5-ui是财经线基于vue2.x的动端组件库组件库，因还未公测所以版本更新比较频繁，请在项目中谨慎使用，如有问题请及时企业微信联系skyinzhang，也欢迎前端小伙伴一起帮助此组件库变得更好。

# 优点

  1. 按需加载组件库（ 即，组件独立打包，单个文件对应单个组件 ）
  2. 引入代码转换的插件

# 准备

在您使用s2h5之前，请您务必先去学习vue及vue组件的相关知识，并了解移动端rem的相关知识

以下5点内容请务必详细了解：

  • props: 传递数据
  • slot: 内容分发
  • $emit: 事件
  • @click.native: 绑定原生事件，在Vue2.x中，为自定义组件绑定原生事件必须使用修饰符
  • rem: 是相对根元素的字体大小变化而变化的css单位，s2h5-ui是通过js动态改变根元素的字体大小实现屏幕自适应。 详情请查看：可伸缩布局

# 全局引入

  1. 在main文件中添加如下

// 引入rem
import remHandle from "@/utils/rem";
remHandle(window, 750);

// 引入组件
import S2H5 from "@tencent/s2h5-ui";
// 组件样式
import "@tencent/s2h5-ui/lib/s2h5ui.css";
Vue.use(S2H5);
  1. utils中rem的计算逻辑代码:
/**
 * * @2019-10-30
 * * @author skyinzhang
 * * @params window window对象
 * * @params designWidth 设计图文档宽度
 * * @describe 可伸缩布局 rem计算方式： 设计稿尺寸 px / 100 = 实际rem 例: 100 px = 1 rem
 */
const remHandle = (window: any, designWidth: number) => {
  let winDom = window.document,
    docEle = winDom.documentElement,
    resizeEvt = "orientationchange" in window ? "orientationchange" : "resize";
  /* 重置font-size的大小 */
  let recalcFontSize = (function refreshRem() {
    let clientWidth = docEle.getBoundingClientRect().width;
    /* 8.55：小于320px不再缩小，11.2：大于420px不再放大 */
    docEle.style.fontSize =
      Math.max(Math.min(20 * (clientWidth / designWidth), 11.2), 8.55) * 5 +
      "px";
    console.log(">>>>>>>>> fontSize: ", docEle.style.fontSize);
    return refreshRem;
  })();
  /* 添加倍屏标识，安卓为1 */
  docEle.setAttribute(
    "data-dpr",
    window.navigator.appVersion.match(/iphone/gi) ? window.devicePixelRatio : 1
  );
  if (/iP(hone|od|ad)/.test(window.navigator.userAgent)) {
    /* 添加IOS标识 */
    winDom.documentElement.classList.add("ios");
    if (
      parseInt(
        window.navigator.appVersion.match(/OS (\d+)_(\d+)_?(\d+)?/)[1],
        10
      ) >= 8
    ) {
      /*  IOS8以上给html添加hairline样式，以便特殊处理 */
      winDom.documentElement.classList.add("hairline");
    }
  }
  if (!winDom.addEventListener) return;
  window.addEventListener(resizeEvt, recalcFontSize, false);
  winDom.addEventListener("DOMContentLoaded", recalcFontSize, false);
};

export default remHandle;

# 按需引入

  1. 安装 babel-plugin-component
yarn add  babel-plugin-component -D
# 或者
npm install babel-plugin-component -D
  1. 修改babel配置文件 .babelrc （注意：vue/cli3更改babel.config.js）

.babelrc 配置 ：

  "plugins": [
    [
      "component", // babel-plugin-component
      {
        "libraryName": "@tencent/s2h5-ui", // 组件库的名字
        "camel2Dash": false, // 是否把驼峰转换成xx-xx的写法
        "styleLibrary": {
          "base": false, // 是否每个组件都默认引用base.css
          "name": "style" // css目录的名字
        }
      }
    ]
  ],

babel.config.js 配置

module.exports = {
  presets: ["@vue/cli-plugin-babel/preset"],
  plugins: [
    [
      "component", // babel-plugin-component
      {
        libraryName: "@tencent/s2h5-ui", // 组件库名
        camel2Dash: false, // 是否把驼峰转换成xx-xx的写法
        styleLibrary: {
          name: "style",
          base: false
        }
      }
    ]
  ]
};
  1. 在main中引入base.less，用于初始化样式、iconfont字体样式引入、主题定义等

// rem
import remHandle from "@/utils/rem";
remHandle(window, 750);

// base 样式
import "@tencent/s2h5-ui/lib/style/base.css";
  1. 页面引入

这里介绍两种使用方式： 在main中按需引入 和 在具体页面或组件中引入

方式一：在main中按需引入：

// rem
import remHandle from "@/utils/rem";
remHandle(window, 750);

// base 样式
import "@tencent/s2h5-ui/lib/style/base.css";

// 按需引入样式
import { Layout, Keyboard, Button, Icons } from "@tencent/s2h5-ui";
Vue.component(Layout.name, Layout);
Vue.component(Keyboard.name, Keyboard);
Vue.component(Button.name, Button);
Vue.component(Icons.name, Icons);
// 也可以使用 Vue.use(组件名) 例如: Vue.use(Layout)

方式二：在具体页面或组件中引入

<template>
  <s2h5-layout>
    <div @click="showKeyboard = true" style="border:1px solid red;">
      {{ amount }}
    </div>
    <s2h5-keyboard
      :is-normal="true"
      v-model="showKeyboard"
      :callback="keyboardHandl"
      ref="kbdemo1"
    ></s2h5-keyboard>
  </s2h5-layout>
</template>

<script lang="ts">
import Vue from "vue";
import { Component, Watch } from "vue-property-decorator";
// 按需引入组件
import { Layout, Keyboard } from "@tencent/s2h5-ui";
@Component({
  components: {
    "s2h5-layout": Layout,
    "s2h5-keyboard": Keyboard,
  }
})
export default class Home extends Vue {
  // 是否显示虚拟键盘
  showKeyboard = false as boolean;
  // 金额
  amount = 0 as number;

  keyboardHandl(val: number) {
    console.log("输入的值为" + val);
    if (val.toString() === "") {
      val = 0;
    }
    this.amount = val;
  }
}
</script>
