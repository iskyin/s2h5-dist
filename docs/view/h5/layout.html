# 布局(Layout)

移动端常布局一般包含页头 、主题内容、页尾。

s2h5-ui中的整体结构采用flex布局，宽高都为100%。

# 按需引入

import { Layout } from "@tencent/s2h5-ui";

@Component({
  components: {
    "s2h5-layout": Layout,
  }
})
export default class Home extends Vue {}

# 代码示例

  1. 自定义头部

    <template>
    <s2h5-layout>

        <!-- 可定制 头部 -->
        <s2h5-navbar slot="navbar" title="布局的标题">

                <s2h5-navbar-back-icon></s2h5-navbar-back-icon>
            </router-link>

        <!-- 中间内容 -->
        <div style="font-size:5.5rem;background-color:rgb(0, 82, 217);color:#fff;text-align: center;">中间内容</div>

# API

注意：s2h5-layout 父组件高度必须为100%，否则会影响页面滚动

# props

参数 说明 类型 可选值 默认值
link s2h5-navbar的跳转链接 String - -
title s2h5-navbar的标题 String - -
show-navbar 是否显示s2h5-navbar Boolean - true

# slot

name 说明
navbar 页头内容
top 上部内容
- 主体内容
bottom 下部内容
tabbar 页尾内容
