# 设计规范-移动端

# 简介

  • 适用于财经系统PM、设计师、前端
  • 设计稿基于通用 iPhone6 750*1334 屏幕尺寸设计
  • 设计稿中文字体：微软雅黑 英文字体：Arial,开发时替换成相应系统默认字体

# 颜色

主色：#457fff — 主要用于头部、按钮、重要性文字和icon

辅色：#0052d9 — 用于个别重点性文字、icon（腾讯蓝）

辅色：#76a7ff — 用于部分按钮、链接文字和icon

辅色：#e8f2ef — 用于底部背景色

辅色：#f0f0f0 — 用于背景色、划分内容区域

辅色：#c9c9c9 — 用于未输入提示性文字、失效按钮、icon分割线等

辅色：#8a8a8a — 用于文字信息名称，描述性文字等

辅色：#343434 — 用于重要级文字信息、内页标题信息

辅色：#ff8484 — 用于移除、删除、错误提示，取消等内容

辅色：#ff5555 — 小面积使用，按钮等个别重要数字

辅色：#79c000 — 用于部分按钮、链接文字和icon（春天绿）
