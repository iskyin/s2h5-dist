# 摄影-16-35F4摄影技巧

# 广角人像技巧

使用广角镜头拍人像的优点：

  1. 可以利用广角镜头的变形特征来修饰拍摄对象的身材。

  2. 可以利用其透视变形的特性来增强画面的张力与主题的冲击力。

拍摄这张照片时，摄影师很靠近模特。只有使用这种拍法，才有办法将模特从画面中凸显出来，才能够展现魅力。模特的脚部位置非常靠近画面底部，这样拉伸的效果就得以很好地发挥。

如何表现广角镜头的张力？

要表现广角镜头的变形张力，最重要的是进行近拍，所谓广角要近，指的是广角镜头要接近目标拍摄。

广角镜头的缺点：

  1. 控制不好变形的话，画面会很奇怪。

  2. 不容易突出主体。

  3. 视角广的镜头要注意保持画面干净，对构图的难度会增加。

使用广角镜头拍摄时，离模特太远会使主体显得不够突出，且带入太多背景也会使画面显得太杂乱等

用广角镜头拍摄时，除了掌握距离以外，取景的角度也是很重要的。其实不论什么镜头，取景的角度都很重要，但在广角镜头上特别明显。

强调画面的广度和张力时，可以由下往上拍摄，这时候画面的张力最强，线条的延伸扩散最具冲击力。

获得干净的画面，则由上往下取景拍摄，这时候可以得到最干净单纯的画面。

使用角镜头时最忌讳摄影师站着，模特也站着，这样很容易就会拍出杂乱的画面，而且照片看起来像是模特站在背景纸前面拍摄的。

由上往下取景，并且搭配大片颜色一致的背景，可以得到很干净的背景。背景干净，拍摄的对象就突出，从而可以很容易彰显拍摄的主题。

利用广角镜头的变形来修饰模特的腿型一般摄影师在拍摄人像时使用广角镜头有一个很大的目的，就是希望可以用广角镜头的变形来把模特的身型拍得更修长，尤其是可以把腿拍得很长。当然这是有诀窍的，用得好的话把五头身的模特拍成九头身都有可能，但是万一用不好，也可能把九头身的模特拍成五头身。

想利用广角镜头把模特的腿拍得很长，前提条件是使用竖构图，由于镜头的特性，横构图会压扁模特，还没拍就失败一半了。

采用竖构图进行拍摄时可以把画面进行三等分，通常摄影师会将模特的头和身体放置在画面中间的位置，用广角镜头拍摄时，这个区域的变形程度是最轻的。我们在拍摄时虽然希望模特的腿变长，但不希望模特的头和身体也跟着变形，因此把头、手和身体放在正中间，可以将其变形程度控制到最小。

接着将模特的腿放置在三等分画面下面的那一等分，广角镜头的变形是由中间开始往上、下、左、右拉伸延长的，所以把腿放置在这个部分，就可以轻轻松松地把腿拍得很长。

上图是使用19mm 广角端拍摄的照片，并利用广角三分法进行构图，很明显将模特的腿修饰得更长了。
