# 快门

快门是摄像器材中用来控制光线照射感光元件时间的装置。

快门速度单位是“秒”。全幅相机的最高快门速度达到1/16000秒。常见的快门速度有：1 1/2 1/4 1/8 1/15 1/30 1/60 1/125 1/250 1/500 1/1000 1/2000等。这里的1是代表1秒，1/2代表1/2秒，1秒往高调整还有2秒,3秒,4秒直到B门（B门是自己控制快门开启的时间长短，在按下快门键时，快门就开启直到放开快门键后，快门才会合上）

相邻两级的快门速度的曝光量相差一倍，我们常说相差一级。如1/60秒比1/125秒的曝光量多一倍，即1/60秒比1/125秒速度慢一级或称低一级。

因为很多时候我们都是 手持拍摄，这里会存在一个安全快门的概念，安全快门通常在1/30s或是1/60s ，相机快门“咔嚓”的时间如果低于1/60s， 便很容易因为手的晃动，而让画面变的很模糊。

一般来说安全快门速度是焦距的倒数，也就是安全快门速度=1/焦距，例如你在使用250mm焦距进行拍摄，他的安全快门速度就是1/250,，快于1/250的快门速度都是安全的，没有太大的失误都是可以完成手持拍摄的，反之则是危险的，有可能出现抖动不清楚的现象，要注意这里的焦距指的是等效焦距。

下面是同环境下高速快门拍摄的图片和低速快门拍摄的图片。

很明显，在低速快门下，出现车灯形成的车轨，但是高速快门下，还可以几乎清晰看到汽车的外形和轮廓。

因为我是使用三脚架拍摄的两张照片，所以车轨这张比较清晰，如果手持的话，第二张照片一定会出现晃动情况，就是因为当时的快门速度已经远远低于安全快门速度。

所以在实际拍摄过程中，大家一定要根据当时的实际情况进行快门的选择，不要盲目的选择拍摄参数。
