# 摄影-焦距

物理学上的定义是指平行光入射时从透镜光心到光聚集之焦点的距离，在单反摄影中指的是镜头中心点到感光器平面的距离（镜片中心到底片或CCD等成像平面的距离）

简单概况：焦距决定了你拍摄照片时候的视角

# 焦距 与 等效焦距

以全画幅和APS-C（残画幅）为例，可以直观看到全画幅的面积明显大于APS-C，所以也很容易理解，在其他条件相同的条件下，配有全画幅传感器的单反成像会大于配有APS-C传感器的单反。

  • 全画幅相机：镜头的焦距 == 等效焦距
  • APS-C相机：等效焦距 == 镜头的焦距 x 焦距转换系数 ( 一般来说尼康和索尼是 镜头焦距 x 1.5 ，佳能是 镜头焦距 x 1.6 ，比如一个50mm的镜头，用全画幅拍摄就是50mm的焦距，但是如果是尼康和索尼的APS-C单反就相当于（50 x 1.5）=75mm，佳能的APS-C单反相当于（50 x 1.6）=80mm )

# 视角

全画幅相机 镜头焦距 与 视角 间的关系图，如下图：

焦距的数字越小，视角越宽， 即取景越宽，画面中物体，人，风景越多，但是每个角色在画面中的面积就越小。

焦距的数字越大，视角越窄， 即取景越窄，画面中物体，人，风景越少，但是每个角色在画面中的面积就越大。

镜头按照焦距分类可以分为以下几种：

  • 24mm以下的焦距可称之为超广角。适合拍摄大场景，表现比较有冲击力的画面。
  • 24-28mm的焦距为标准广角。一般用于风景拍摄，比较常用的焦段，也容易驾驭。
  • 28-35mm的焦距为一般广角。人文必备，多数人拍摄人文首选焦段。
  • 35-70mm的焦距为中焦。人文，人像摄影。
  • 70-135mm的焦距为中长焦。适合人像。
  • 135-400mm的焦距为长焦。长焦适合一些例如体育摄影等场景。
  • 400mm以上的焦距为超长焦。“打鸟”镜头。

# 变焦镜头 与 定焦镜头

# 变焦镜头

变焦镜头一般在机身已有 XX-XXX mm, 比如：24-70mm （指镜头覆盖了24mm到70mm焦段），是一支2.92倍的变焦镜头（倍数计算：70÷24=2.92）

特点：

  • 可实现焦距在一段范围内的调整
  • 大多数专业变焦镜头的最大光圈为f / 2.8

# 定焦镜头

定焦镜头以一般只写 XX mm, 比如：50mm （指镜头焦距为50，只能使用前后移动才能改变焦距）

特点：

  • 定焦镜头比变焦镜头更小，更轻，更紧凑
  • 对焦更快
  • 提供f / 1.8或甚至f / 1.2的大光圈

定焦镜头一般有：35mm、50mm、85mm

  • 35mm焦距段--街头摄影常用镜头

35mm焦距是摄影记者和街头摄影师的最爱，因为它适合拍摄更加宽广画面的镜头。它的宽度足以用于狭窄的街道或景观，但也不会太宽，导致您的画面变形失真。35mm也是是大多数智能手机的焦距。

  • 50mm焦距段--标准镜头

在全画幅相机中，50mm镜头是最接近于人眼的视野（在半画幅相机（APS-C）上，35mm焦距的镜头是最接近人眼视野的镜头）。由于这个原因，50mm焦距是最受欢迎的定焦镜头之一。通常50mm有相当大的光圈（1.8,1.4甚至1.2）; 所以它非常适合拍摄分离背景或在低光照条件下场景。

虽然50mm焦距镜头是一款梦幻般的万能镜头，但它有时缺乏其他镜头的“魅力”。例如，如果想要把人像拍摄的比较瘦，一种方法是使用较长的焦距段，如85mm甚至135mm，因为它们会使脸部略微变细，并且它们会提供更好的背景模糊效果。

  • 85mm焦距段--人像镜皇

适用于拍摄全身，腰部以上和近距离特写。在拍摄近距离特写时，这时必须小心，因为会出现面部特征的扭曲。在室内拍摄时，85mm镜头非常适合拍摄膝盖以上的人像和头部特写，但对于更近距离的人像特写，135mm将是更好的选择。
