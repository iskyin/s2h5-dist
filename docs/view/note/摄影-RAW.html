# 摄影-RAW

即，RAW格式文件，RAW的原意就是“未经加工”。

RAW图像就是CMOS或者CCD图像感应器将捕捉到的光源信号转化为数字信号的原始数据。RAW文件的文件后缀是.nef（尼康），.cr2（佳能），.arw（索尼）。

前面我们说的白平衡，因为很多人都会采用RAW格式储存文件，所以如果前期白平衡没有校准成功的话，可以后期进行调整来修正白平衡，RAW格式文件在修正过程中基本上可以达到无损，但是JPEG文件的调整空间极其有限，而且调整后也会造成图片信息的损失。

白平衡是摄影中很重要的一个概念，但是很多人在拍摄过程中会忽略这个因素，包括我也是，更多时候都会选择自动白平衡，后期再进行修正，但我个人还是建议如果一个新手来拍摄的话，如果有能力校准，尽量校准白平衡，这样你的后期过程就有了一个起始点，你再进行后期创作的过程中，也有了参照标准。
