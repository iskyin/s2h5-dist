# 摄影-光圈

# 光圈

光圈是一个用来控制光线透过镜头，进入机身内感光面光量的装置，它通常是在镜头内。对于已经制造好的镜头，我们不可能随意改变镜头的直径，但是我们可以通过在镜头内部加入多边形或者圆形，并且面积可变的孔状光栅来达到控制镜头通光量，这个装置就叫做光圈。

一般用F来表，F=镜头焦距/镜头有效口径直径。

# 景深

景深指在摄影机镜头或其他成像器前沿能够取得清晰图像的成像所测定的被摄物体前后距离范围，在摄影中我们要记住的是，景深越浅，背景虚化越强，景深越深，背景虚化越弱。而大光圈就带来了浅景深，小光圈则会造成深景深。

F后面的数值越小，光圈越大，进光量越多，画面比较亮，景深较浅，主体后面的背景虚化越强；

F后面的数值越大，光圈越小，进光量越少，画面比较暗，景深较深，主体后面的背景虚化越弱。

一般来说，大光圈适合拍摄凸显人物主体等场景， 但是当拍摄团体照等不适合使用，因为会造成边缘人物虚化，无法得到清晰的图像，小光圈适合拍摄尽可能保留大场景中细节的图片，例如旅游时候的团体合照，另外如果 在夜晚在小光圈下拍摄路灯等光源，还可以得到“星芒”效果。
