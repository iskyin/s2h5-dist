# 摄影-构图

# 目的

  • 首先它要主动去引导你的视线先去看哪里，看什么内容，是最重要的
  • 其次画面里有顺序安排，凸显哪个是主体，哪些是衬托的视觉元素
  • 构图构得好，能带出画面的情绪，当然这也是最难做到的

# 作用

  • 主动引导观看图片的人群
  • 表明主次
  • 主动表达情绪

# 人眼 vs 相机

# 人眼

  • 两只，看东西是一直在动的
  • 眼睛在环顾四周时看到的是动态的成像

# 相机

  • 一只，抓拍那一瞬间
  • 只拍摄一幅静桢

意大利的比萨斜塔旁边有个很有趣的景象：很多人都做出推的姿势。然后拍下和斜塔的合照，看起来仿佛就是这个人在推倒或支撑住了比萨斜塔。所以没 3D 感，善加利用的话拍出的照片是非常有趣有创意的。

除了平面之外，最容易让人对物体大小比例产生错觉，人一直在动，看东西是立体的，如果你是拍照，你可能会用跟人的透视不一样的，或许是个广角，或许中长距离，85 mm，它会让人的背景和大小比例产生错觉。

# 拍照 vs 画画

  • 画画是加法，是作者对脑海中所想象的事物的表达，是从一张白纸上开始的
  • 拍照是减法，是对真实场景的纪录。

但是你眼睛里看到的东西，相机全都呈现时却与你看到的不同，所以这时候你需要一个合适的构图来将你想表达的东西表达出来。

为了可以进一步去思考怎么样构图更好，大家可以想想画画和摄影之间的区别，空白画布里，你想画面里出现什么，你得要画进去。相对来说，摄影比较容易，你相机对着哪里，只要曝光充足，可以减去一些不想要的视觉元素，画面才会干净，有趣，摄影呢是减法，你只需要考虑什么是可以减去的，便可以增加画面冲击力。

著名的马格南摄影师 Robert capa 曾经说过：如果你的照片不够好，是因为你靠得不够近。

所以刚刚提到摄影靠减法，下面这张是原图，拍一个著名音乐家，原图体现了很多细节，比如钢琴底下的脚什么的，但后来杂志刊登时，他裁掉了一些地方，最后的画面就成为很经典的一张。

第二张张黑白就是最后刊登出来很经典的那张，那个人是主角，虽然只占画面里的 10%，钢琴很大，但我们注意力还是集中在这个人身上，刚请似乎成为了很多几何线条，人变成了钢琴的延伸。

减少的视觉元素反而让其有了更大的体现能力。

# 有前到后

构图另外一个重点是对比，其中一个很重要的就是前后对比，主体前面叫前景，通常拍照都会有背景，但不一定有前景，其实可以多留意一下怎么在你的主体前面摆一些前景，挡一些东西，让它成为前体，这样画面会更多趣味，有了前后对比，画面也更多立体感。

下面第一张张图有两种对比，前后对比和大小对比，大小对比的意思其实就是在同一个画面里利用大小或者一前一后其中一个来衬托主体。除了前后对比，也有大小对比，同样也是增加画面立体感的一种方法。特别广角镜头越靠近你，东西变得越大，越远越小。

得到大小对比大致有三种方法：

  • 利用拍摄对象自身的面积和体积的差异产生对比得到效果
  • 利用镜头近大远小的透视变化得到对比效果
  • 两点结合起来，把本来很小的物体夸大，制造出反常规的大小对比，例如刚刚提到的比萨斜塔

下面第一张就是用长镜头拍的，有压缩空间的特点，会把你的主体和背景拉得更近，而广角呢反而会把两者夸张化，反而变得更远，这就是一种透视感特点。

如果用长焦，你站远一点拍，主体和用广角镜头呈现出来的或许一样，背景呢就会用大小变化，因为我刚有说，广角是靠得越近，东西变越大。

第二张是超广角，18 mm，21 mm，你站靠近你的拍摄主体，还是那句话，如果你拍得不够好，就是站得不够近。你必须要站得很近，才会有视觉冲击力，因为比例关系和背景被夸张化了。

正常视角下拍摄大小对比不明显的物体时，如果使用了广角镜头，就会立刻夸张大小对比的效果，在这些照片中，离镜头比较近的物体明显被夸大。只要在能够清晰对焦的范围内，用广角镜头贴得越近，近景与远景就对比越夸张。

下面这组是著名时尚大师 Steven Miesel 拍得，通常时尚片里很少用超广角，但他破例用了，反而显得特别有视觉冲击力。大家可以去多看看他作品，同时也是个大帅哥。当时和 Mario Testino 一样都是《VOGUE》杂志的御用。

但其实这种拍法很容易畸变，尤其是位于前景部分的被摄物。所以，如果用超广角来拍人物，要注意身体比例容易走形。

如果你想拍经典肖像照时，越靠近越走形，所以中距离选择 85 mm，105 mm 比较合适。拍全身的话，超广角会让人看起来更高，但身体比例容易走形，有利有弊吧。

# 抓大等小

常大家扫街的时候可能会用得比较多。

抓大指的其实是一个比常规更大的物体，例如路边巨幅的广告牌，或是巨型雕塑，这些物品本身就有夸张的效果，是创作的好素材。

等小就指的是只要等到一个合适的小的陪体出现，就可以创作出一幅有着大小对比技巧的作品啦。不过这种等待需要耐心和精神的集中，就像渔夫等待者上钩的鱼。

下面这是作者在美国洛杉矶的街拍，当时经过这家店，觉得窗口的那些画很有趣，他就在那里等着，他就像个渔夫一样，他知道不用一会儿就有很多路人走过，他会等到有意思的画面出现然后抓拍下它。

# 疏密对比

Elliot Erwitt 的这张图利用疏密对比讲述了他想表达的主题，疏与密让整张图很鲜明的分成两部分。

Elliot Erwitt 是玛格南很棒的摄影师，他的片子特别有内涵，第一眼你看就会喜欢，越看多几眼你会更加喜欢。

这种内涵片子可以说真的可遇而不可求。他就喜欢在博物馆里呆着，看到想要的画面就抓拍，他就是渔夫型摄影师，博物馆就是他的鱼塘，等着鱼出现就抓住。

# 颜色

红绿之间的对比，虽然绿占的比重很大，但我们注意力却会跑去红那里，因此带出一个从左到右的视线转移，下次你可以分析一下自己的作品，看是从左到右呢还是从上到下这带来的情绪也会完全不一样。

# 线条

这一张“一心一意”，很清晰的相片常常会引起理性的反应，你会很想去读那个字，而第二个版本我把后面给虚化了，这样一来看不到字，你的反应会情绪化，艳红色和深咖色，他们之间的分割形成一个横线条，视觉上从左到右，就像画面里那个女孩，也是从左到右在行走。

线条呢会带出动感，特别是斜线条，从左下到右上，或从右上到左下，感觉都不一样，特别值得注意的是两点成一线，后面会出现光斑或明暗对比，会引导你的视线从左到右或从上到下。

好像这张图里的地平线它不是横着，有点斜着向上移动，所以带来点动感，你们的相片竖的和横的，如果地平线不是和竖或横平行的话，就会带来动感。

从左下到右上，通常这样的斜线条会带来比较正能量的这种情绪反应

# 框架式构

良好构图的一个诀窍就是寻找适合拍摄主体的框架，比如一扇窗或者一个门。框架式构图能把观众的视线引向框架内的景物，突出主体。形成一种框架可以营造一种神秘的气氛，就好像是偷窥。

# 将最有穿透力的那只眼睛放在画面的正中间

拍摄人物有个特别有用的技巧，你要先留意对方的眼睛，特别是最靠近那只，你对焦的话就要对焦在最靠近的那只眼睛上，放在正中间的话，这样一来视线会更加有穿透力。

# 左右对称

注意乔布斯那张图，摄影师他刻意把人的脸放在左右对称的位置，左右对称这个方法我在教学生拍摄时，是要求他们首先要了解和掌握的，因为左右对称拍得多了，你会对人的脸非常敏感。

见到一个人第一眼就可以看得出它左脸好看还是右脸好看，从而在前期无论是发型、妆容还是模特的摆姿和选取的角度上都可以扬长避短。

把人脸拍到左右对称呢，听起来蛮容易，实际操作还有点难，我 10 个学员里有 9 个起码第一次拍摄中都没办法把人脸拍到左右对称，都会偏左或偏右一点。如果你发现你的模特如果只把一边脸偏向你，说明那半张脸是她比较满意的，觉得拍出来会好看

# 瞬间美

接下来，我说瞬间美 Decisive Moment，也就是玛格南著名摄影师布列松提出的“决定性瞬间”。例如这张，鸽子飞向空中，刹那间怎么构图，在抓拍那一刻曝光也准确，那么就达到了我们所说的“瞬间美”。

再例如，下面前两张这张也是拿手机拍的，让手机靠近地面，刚好一个人走过抓拍到，早一刻迟一刻都不会有。
