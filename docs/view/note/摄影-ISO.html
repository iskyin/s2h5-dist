# 摄影-ISO

# ISO

ISO就是感光度，感光度是衡量底片对于光的灵敏程度，对于单反相机来说，它是一个可以调节的数字，数字越小，拍出来的照片越清晰，数字越大，拍出来的照片越模糊。

ISO可以设置为自动，这种设置下ISO会根据其余拍摄参数以及环境进行自动调整，也可以关闭自动，根据实际需要，固定ISO的数值大小来进行拍摄。一般来说，为了保证拍摄成功，ISO都采用固定设置，锁定ISO的数值大小，修改其余参数来拍摄。

# 噪点

是指CCD将光线作为接收信号接收并输出的过程中所产生的影像中的粗糙部分。即，在画面中看到的一些本不应该存在的颗粒。

# 低感光度

ISO800以下为低感光度。

在这一段，拍摄的照片可以细腻，噪点少。环境允许的情况下，为了照片质量尽可能使用低ISO。有的环境下，三脚架的使用可以降低ISO。

# 中感光度

ISO800~6400属于中感光度。

在这一段，需要根据你拍摄的照片的用途，在能接受的噪点程度下，中ISO可以使摄影师采用手持拍摄，如果环境光很弱，也可以保证拍摄成功。

# 高感光度

ISO6400以上是高感光度。

在这一段，成片噪点明显，一般是在恶劣环境下，而且是保证拍摄成功的重要性大于拍摄照片的质量。

感光度即是对光线的敏感程度，在相同光圈以及快门速度下，感光度越高，照片也就会越亮, 如下图所示，随着ISO自100提升至8000，照片将逐渐变亮，越来越能分辨出照片里面中的元素。

在这里很确定的一个观念是，光线充足的时候，要使用低ISO，但光线不充足的时候，若想要纯净的照片，你一定要使用三脚架等固定相机的设备，搭配较低ISO来拍摄，若你没有这些设备或不具备使用条件的情况下，你也只能通过提升ISO，保证画面有充足的曝光量以及不抖动。
