# 摄影-透视

透视说的直白一点，就是远小近大。透视和焦距并没有太大的关系，透视是和拍摄距离有关的。

一般来说拍摄距离越近，透视感越强烈；拍摄距离越远，透视感越不强烈。

广角镜头焦距短，需要表现相同大小的拍摄主体时，摄影师要做到拍摄距离近，透视感会强；反之，长焦镜头焦距长，需要表现相同大小的拍摄主体时，摄影师要做到拍摄距离远，透视感会弱。

如下图，中间的扶手，明明是平行的两根铁管，但是看起来确是近处很大，远处就像聚拢汇聚到一点。

同样的情况也会出现在你拍摄建筑物的时候，明明是矩形的建筑物，在你的拍摄下，会变成梯形，下面宽，上面窄，这就是因为透视。

人像摄影中，透视至关重要，广角镜头产生的透视感太强，会让你要拍摄的妹子出现“大鼻子”效果，如果选择适当的长焦，会削弱透视效果，会显得妹子脸小一些，所以50mm和85mm一般很适合人像拍摄，当然这里还会涉及到光圈和景深
