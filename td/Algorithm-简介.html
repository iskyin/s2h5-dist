# 复杂度分析

表示算法的性能

  • 通常看最差的情况
  • 算法运行的上限
  • 常熟不重要
  • 复杂度描述的是随着数据规模 n 的增大，算法性能的变化趋势
  T = 5000n + 100000
  或
  T = 5n +10
  此两种函数的算法复杂度都为： O(n);

  T2= 2n²
  此种算法： O(n²)

  O(n) <  O(n²)

  注意：复杂度表示的是随着数据规模n的增大，算法性能的变化趋势
